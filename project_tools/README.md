# project_tools

Набор вспомогательных модулей для ноутбука проекта:

- `dataset.py` — классы и вспомогательные функции для датасета.
- `eda.py` — валидация COCO, графики и визуализация боксов.
- `fcos.py` — запуск обучения, оценки и инференса FCOS.
- `yolo.py` — обучение, валидация и инференс YOLOv8.
- `video.py` — запись видео-инференса через OpenCV.
- `metrics.py` — таблица итогового сравнения моделей.
